Getting Started with Snackii API

Documentation for developers integrating Snackii into external applications.

Base URL

https://api.snackii.rp/v1

Authentication

Include your API key in the header:

Authorization: Bearer YOUR_API_KEY

Example: Get Food Recommendations

GET /recommendations?preference=vegetarian&budget=10
Response: {
  "recommendations": [
    {
      "id": 1,
      "name": "Vegetarian Rice Bowl",
      "location": "Canteen A",
      "price": 5.50,
      "rating": 4.5,
      "distance_from_you": "200m"
    }
  ]
}

Rate Limiting

Standard tier: 1000 requests per hour per API key

Support

For API issues, contact api@snackii.rp

About Makan@RP

Our Mission

Makan@RP is an AI chatbot solely to solve the indecisive choices we face when choosing a place to eat in RP, with the all now AI-Chatbot, you'll be given all food options near you!

What We Do

Makan@RP provides a smart AI assistant called Snackii that:

  • Recommends food options based on preferences and budget
  • Shows real-time operating hours and availability
  • Learns from user feedback to improve suggestions
  • Supports multiple dietary requirements
  • Helps discover new food spots on campus

Why Makan@RP?

Makan means "eat" in Malay and Indonesian, reflecting the food-centric nature of our service. @RP signifies our dedicated focus on the Raffles Institution (RP) community.

Snackii - The Smart Chatbot

Snackii is our AI-powered chatbot trained on comprehensive campus food data. It understands natural language queries and provides contextual, helpful recommendations in real-time.
